# Mission Atlantic RISK Analysis

## Introduction

Script for performing the Mission Atlantic RISK Analysis. It has been modified from the ODEMM ('Options for Delivering Ecosystem-Based Marine Management') - approach.

By Debbi Pedreschi

## Directory structure

```
MissionAtlantic-ODEMM-Analysis/
├── analysis (empty)
├── data/    (empty)
├── docs/    (empty)
├── product/ (empty)
└── R/
     └── MA RISK analysis script.R
```

* **analysis** - RMarkdown notebooks
* **data** - Raw and derived data
* **docs** - Rendered reports
* **product** - Output product files
* **R** - Reusable R scripts/code
